# Hospital-system# Hospital Management System 🏥

## Description
This is a simple Hospital Management System project built using HTML and JavaScript.  
It allows adding patient information and displaying it in a table.

## Features
- Add patient name
- Add patient age
- Add disease
- Display patients in a table

## Technologies Used
- HTML
- JavaScript

## How it works
User enters patient data and clicks "Add Patient", then the data appears in the table below.

## Project Purpose
This project was created for learning purposes to practice DOM manipulation using JavaScript.
